Features

Scala 3 has a simple Python-like syntax, which can be learned quickly by Java programmers.

JVM

Scala 3 is powered by the JVM, the most proven platform for deploying server-side applications.

Ecosystem

SScala 3 interoperates with libraries and code in Java & Kotlin, and can benefit from JVM tooling.

Type-inference

Scala 3 has advanced type inference, retaining the ease of dynamic programming with the safety of static types.

Data Classes

Scala 3 has data classes and enums with rich features that eliminate boilerplate and code generation.

Generics

Scala 3 has a powerful generics system, allowing reuse of data structures and code across data types.

Functional

Scala 3 has a functional core, with first-class functions, immutable data, pattern matching, and type classes.

Code Derivation

Scala 3 has code derivation, a powerful feature that can derive serialization codecs and other code for you.

Meta programming

Scala 3 has advanced metaprogramming, allowing you to reduce boilerplate and improve performance.
